# E-Commerce Sales Analysis Dashboard 📊

## Project Overview
This project focuses on transforming raw e-commerce data into actionable business insights. Using **SQL** for data extraction and **Power BI** for visualization, I identified key trends in revenue growth and regional performance.

## Tech Stack
* **Database:** SQL (PostgreSQL/MySQL)
* **Visualization:** Power BI
* **Data Cleaning:** Power Query

## Key Insights
* **Revenue Growth:** Identified a 15% month-over-month increase during Q4.
* **Regional Performance:** The North region accounts for 40% of total sales but has the lowest profit margin.
* **Category Focus:** Technology products drive the highest revenue, while Office Supplies have the highest order volume.
* **Discount Strategy:** Analyzed the correlation between discount tiers and net profit, identifying that heavy discounting in the "Government" segment resulted in a 5% margin dip.
